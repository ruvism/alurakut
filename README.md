# Você está de volta aos anos 2000!

Bem-vinde ao meu Orkut, ou melhor, Alurakut, desenvolvido juntamente ao time da Alura em uma imersão de Reactjs em julho de 2021. 

Como as coisas não andam muito legais atualmente, vamos voltar aos anos 2000 e reviver tempos mais fáceis em que a preocupação era olhar o orkut e bater papo no msn. 

Para acessar o projeto basta [clicar neste link](https://alurakut-ruvism.vercel.app/)

Deixe seu scrap e me add por lá xD

## Instruções

Para rodar o projeto localmente, faça o `git clone`.

Na pasta raiz, rode no terminal `npm run dev`. 

A visualização é feita no [localhost](http://localhost:3000/).
